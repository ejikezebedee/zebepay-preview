# Zebepay Product Overview

## Product Positioning

Zebepay is a private commercial Nigerian payment-gateway, wallet, and banking
infrastructure source-code foundation. It is prepared for controlled buyer
review and private commercial delivery.

## Buyer Value

Zebepay gives qualified buyers a serious foundation for:

- Wallet and account workflows.
- Customer banking experience.
- Admin operations review.
- KYC workflow.
- Transfer review and risk controls.
- Audit logs.
- Notification outbox.
- Provider integration surfaces.
- PostgreSQL production-readiness validation.

## Full Private Package Scope

The private source package is intended to include:

- Customer app.
- Admin app.
- Backend API.
- Shared domain package.
- Database schema and PostgreSQL support.
- Buyer handoff and setup documentation.
- Deployment and API documentation.
- Commercial release package and checksum.

## What This Public Preview Does Not Include

This public preview does not include:

- Full customer app source.
- Full admin app source.
- Backend API source.
- Shared package source.
- Database migrations.
- Buyer ZIP.
- Private release audits.
- Commercial decision records.
- Secrets, tokens, env files, logs, or machine-specific files.

## Compliance Boundary

Zebepay is software source code only. Buyers must complete licensing review,
regulated provider onboarding, KYC/AML setup, legal review, compliance review,
production security audit, hosting, monitoring, and final go-live approval.
