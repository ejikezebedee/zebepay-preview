# Zebepay Code Preview

## Purpose

This file explains the private source-code package without exposing
implementation files in the public preview repository.

## Private Source Areas

Approved buyers may review the private repository after access conditions are
met. The private source package is organized around:

- `apps/customer-web` for the customer-facing wallet and banking experience.
- `apps/admin-web` for back-office operations.
- `services/api` for backend API and banking domain services.
- `packages/shared` for shared types and validation.
- `db` for schema and database support.
- `docs`, `sales`, and `release` for buyer delivery documentation.

## Public Preview Boundary

This public repository intentionally contains no implementation source files,
database migrations, env files, logs, secrets, buyer ZIP, or private release
records.

## Review Process

Private source access is only granted after:

- Buyer approval.
- Payment confirmation.
- License terms accepted.
- Refund policy accepted.
- Support scope accepted.
- Buyer GitHub username or organization confirmed.
- Delivery event recorded.

## Security Note

Do not request or share secrets in public issues. Do not post bank credentials,
API keys, BVN/NIN values, private customer data, or confidential documents in
this repository.
